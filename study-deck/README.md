# Study Deck

A dependency-free, responsive HTML/CSS/JavaScript study application generated from the supplied questionnaire PDFs.

## Features

- Five subjects and per-topic practice
- 10-question, 20-question, or full-topic sessions
- Learn mode with immediate feedback
- Exam mode with delayed results
- Incorrect-answer reinforcement: missed questions return later until corrected
- Mixed-subject practice and mistake review
- Local progress and mastery tracking
- Keyboard controls and optional interface sounds
- Mobile-responsive layout
- No framework, server, build step, or external dependency

## Run locally

Open `index.html` in a modern browser.
